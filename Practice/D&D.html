Top-Right Links

Register

You can register for an account on this page. Currently a secret key is needed for registeration.

Login

You may login your account for submissions of your program.

Profilie

This will be shown only if you login, you can edit your personal information here.

Site Index

Submit

After finishing your program, you can submit it for judge through this page. You need to fill in problem ID and language you used, then you can submit your code by either posting it or uploading the file.

Status

All the verdicts will be shown here. The status, CPU time as well as memory usage will be reported for every submission, except thos ought not to be showned during the very end of the contest, when the scoreboard is freezed.

Problem Archive

Here you can find the problems of this site that you can try to solve.

Contest

Problem Set

You can browse the problems of the currently running contest, if any.

Scoreboard

Here you can see the ranklist and status of participants of the currently running contest.

Archive

Here you can see all scheduled contests including the past and upcoming ones.

Admin

If you are an admin, after login, you will see an additional section.

Contest Panel

Here you can schedule for a contest as well as modify the information and problem sets for the contest, e.g. when should the scoreboard freeze before the end of contest.

Problem Panel

Here you can create a new problem or edit an existing problem, editor embedded is Fckeditor.

Note that aside from these additional functions, the admins are allowed to see all the status of the submission and scoreboard during it is freezed. Besides, the problems that is not added to Problem Archive will not be shown and cannot be submitted for non-admin accounts.
